首先，创建自己的网站分为动态和静态两种

**动态网站**：动态网站是指需要服务器端运行的网站，用户访问时需要服务器端处理，生成动态页面。动态网站通常使用PHP、Node.js、Python等后端语言编写，可以部署在Heroku、AWS、Azure等云平台。

流程基本可以概括如下：

1：准备好一个域名。

2：购买一台服务器，在阿里云和腾讯云等平台购买,个人博客的话轻量级服务器就足够了，注册和登录后，完成实名认证（个人 / 企业均可），进入「轻量应用服务器」控制台，点击「新建」

3：服务器基础配置

进入服务器实例详情页，记录公网 IP、管理员密码（忘记可在控制台重置）；

开放端口：在「防火墙」标签页，添加规则开放 80、443、8888 端口（宝塔面板必备）；

重置 root 密码（可选，确保登录安全）。

4：安装与配置宝塔面板

方式 1：一键镜像安装（推荐，新手首选）

进入实例「应用管理」页签，找到「宝塔 Linux 面板」，点击「访问」；

复制面板登录地址（如http://公网IP:8888/随机字符串）、用户名、密码；

浏览器访问登录地址，首次登录需绑定宝塔账号，按提示完成初始化；

面板设置：修改默认用户名、密码、面板端口（建议改为非 8888 的常用端口，提升安全）。

方式 2：手动安装（无镜像时使用）

用 SSH 工具（如 FinalShell、Xshell）连接服务器：

主机：公网 IP，端口：22，用户名：root，密码：重置后的 root 密码；

执行安装命令（以 Ubuntu 系统为例，CentOS 需换对应命令）：


运行
```
wget -O install_panel.sh https://download.bt.cn/install/install_panel.sh && sudo bash install_panel.sh ed8484bec
```

输入y确认安装，等待 5-10 分钟，记录面板登录信息。

五、搭建博客环境与部署

宝塔面板左侧「软件商店」，搜索安装「LNMP」套件（Nginx+PHP+MySQL，适合 WordPress）；

左侧「网站」→「添加站点」，填写：

域名：你的博客域名（如xxx.com，需先解析到公网 IP）；

数据库：选择 MySQL，设置用户名、密码；

其他默认，提交。

安装博客程序（以 WordPress 为例）：

宝塔「软件商店」搜索「WordPress」，一键安装到站点目录；

访问```http://你的域名/wp-admin```，按提示完成博客配置（设置标题、账号、密码）。

六、域名解析与 ICP 备案

域名解析：进入域名服务商控制台，添加 A 记录，将域名指向服务器公网 IP；

ICP 备案：腾讯云 / 阿里云控制台进入「备案」模块，按提示提交资料（域名、身份证、服务器信息），等待审核通过后，博客即可正常访问。

七、安全与优化（可选）

宝塔「安全」模块：开启防火墙、暴力破解防护，设置 IP 白名单；

配置 HTTPS：宝塔「网站→SSL」，申请免费 Let's Encrypt 证书，强制 HTTPS 访问；

博客优化：安装缓存插件（如 WP Rocket）、优化图片，提升访问速度。

阿里云通用注意事项

配置与腾讯云一致，镜像同样可选「宝塔 Linux 面板」，购买时注意地域与时长；

阿里云秒杀款（38 元 / 年）需每日 10 点 / 15 点抢购，限购 1 台；

备案流程与腾讯云一致，优先选官方备案通道，确保合规。

常见问题与避坑

宝塔面板无法访问：检查防火墙是否开放 8888 端口，确认服务器公网 IP 正确；

博客访问慢：选就近地域服务器，开启 CDN（腾讯云 / 阿里云免费 CDN）；

环境冲突：宝塔安装前确保系统纯净，无其他 Web 环境（如 Apache、Nginx）

**静态网站**：静态网站是指不需要服务器端运行的网站，所有的内容都是预先写好的，用户访问时直接从服务器上下载静态页面，不需要服务器端进行处理。静态网站通常使用HTML、CSS和JavaScript等前端技术编写，可以部署在GitHub Pages、Netlify、Vercel等平台上。

流程基本可以概括如下：
1. 注册账号
GitHub：```https://github.com```（免费，存代码）
Cloudflare：```https://dash.cloudflare.com```（免费，托管 + 加速）
域名：阿里云 / 腾讯云买一个
1. 本地安装 Hugo（Windows/macOS）
Windows
下载：```https://github.com/gohugoio/hugo/releases```
（选 hugo_extended_xxx_windows-amd64.zip）
解压到 C:\hugo
把 C:\hugo 加入系统 环境变量 PATH
打开 CMD 验证：hugo version（出现版本号即成功）
macOS
运行
```brew install hugo```
```hugo version```
二、本地创建 Hugo 博客（5 分钟）
初始化站点
```
hugo new site myblog
cd myblog
```
安装主题（以 Ananke 为例）
运行
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
1. 修改配置 hugo.toml
toml
baseURL = "https://你的域名.com"
languageCode = "zh-CN"
title = "我的博客"
theme = "ananke"
1. 写第一篇文章
运行
hugo new posts/hello-world.md
打开 content/posts/hello-world.md，用 Markdown 写内容。
1. 本地预览
运行
hugo server
访问：http://localhost:1313（没问题再下一步）
三、上传到 GitHub（5 分钟）
1. GitHub 新建仓库
仓库名：myblog（随便）
公开 / 私有均可
不勾选 README、LICENSE、.gitignore
1. 本地推送到 GitHub
运行
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/你的用户名/myblog.git
git push -u origin main
四、Cloudflare Pages 自动部署（10 分钟）
1. 登录 Cloudflare → Workers & Pages → Create application
2. 选择 Pages → Connect to Git → 授权 GitHub
3. 选择你的 myblog 仓库 → Begin setup
4. 构建设置（直接抄）
Framework preset：选 Hugo
Production branch：main
Build command：hugo
Build output directory：public
Environment variables（必加，否则 Hugo 版本不对）
名称：HUGO_VERSION
值：0.123.0（用最新稳定版）
点击 Save and Deploy
1. 等待部署（1-2 分钟）
成功后会给你一个测试域名：https://myblog.pages.dev（国内访问一般，后面绑自己域名）
五、绑定自定义域名（国内秒开，关键）
1. 把域名 DNS 托管到 Cloudflare（必须）
Cloudflare → Add a Site → 输入你的域名 → 选 Free Plan
Cloudflare 给你 2 个 NS 地址（如 xxx.ns.cloudflare.com、yyy.ns.cloudflare.com）
去你的域名服务商（阿里云 / 腾讯云）：
找到域名 → 管理 → DNS 服务器 / Nameservers
删除原有 NS，填入 Cloudflare 给的 2 个
保存（生效 5-30 分钟）
2. Cloudflare Pages 绑定域名
进入 Pages 项目 → Custom domains → Set up a domain
输入你的域名（如 你的域名.com）→ Continue
Cloudflare 自动生成 CNAME 记录（确认代理是 橙色云）
点击 Activate domain
等待 1-5 分钟 → 显示 Active 即成功
3. 验证
浏览器打开 https://你的域名.com → 国内秒开、HTTPS 自动生效
